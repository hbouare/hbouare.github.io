Toolkit Python pour la gestion de secrets dans OpenShift. Webhook event-driven, bibliothèque client hvac, CronJobs Kubernetes pour rotation automatique des secrets.

Solution complète remplaçant HashiCorp Vault par OpenBao, avec une intégration transparente dans les pipelines GitLab CI/CD existants.
